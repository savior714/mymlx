# Multi-Agent Tooling Policy

## 목적

본 문서는 멀티 에이전트 환경에서의 도구 사용 일관성과 충돌 방지 운영 정책을 정의한다.

## SSOT

- 전역 운영 규칙: `PROJECT_RULES.md`
- 에이전트 행동 지침: `AGENTS.md`
- 핵심 설계 결정: `docs/CRITICAL_LOGIC.md`

## 정책 범위

- 에이전트 간 도구 사용 원칙
- 동시 수정/충돌 방지 규약
- 상태 동기화 및 문서 반영 원칙

## 비고

기능/API 계약은 `docs/specs`에서 관리하고, 운영 정책은 `docs/policies`에서 관리한다.
